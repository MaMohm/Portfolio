À propos

Mon parcours d'apprentissage

Je suis Marwan Mohammad, actuellement en formation pour devenir développeur web. Passionné par la technologie et la programmation, j'apprends les fondamentaux du développement web avec HTML, CSS, JavaScript et Python.

Mon objectif est de maîtriser les technologies web modernes pour créer des sites web fonctionnels et esthétiques. Ce portfolio documente mon apprentissage et présente mes premières réalisations dans ce domaine passionnant.

Niveau actuel : Débutant avec bases solides en HTML/CSS et apprentissage progressif de JavaScript et Python.

Mon apprentissage

Découvrez les technologies que j'apprends et mes objectifs de formation.

HTML & CSS

Apprentissage des bases de la structure web avec HTML5 et de la mise en forme avec CSS3. Création de pages web responsives et modernes.

JavaScript (En cours)

Étude des fondamentaux de JavaScript pour rendre les pages web interactives. Manipulation du DOM et événements utilisateur.

Python
